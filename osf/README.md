# Replication Materials: *Quantitative Voice Analysis Reveals Context-Dependent Expression of Personality Pathology*

---

## Overview

This repository contains the analysis-ready datasets, Stan models, and R scripts needed to reproduce all statistical analyses reported in the manuscript and its Supplementary Materials. The preprocessing pipeline (EMA import, merging, quality control, participant exclusions) has been applied prior to archiving; the repository starts from cleaned, analysis-ready data.

## Data Files

### `data/processed/audio_required_columns.csv`

Long-format acoustic feature file. One row per participant × assessment period (357 rows total: 119 participants × 3 timepoints). Columns: `timepoint` (BASELINE / PRE / POST), `ID` (anonymous participant code P001–P119), `F0 mean Hz /a/`, `F0 mean Hz /i/`, `F0 mean Hz /u/`, `NNE /a/`, `NNE /i/`, `NNE /u/`. Features were extracted from sustained cardinal vowel phonations using BioVoice software.

### `data/processed/pid5_required_columns.csv`

Long-format EMA file. One row per participant × EMA occasion (3 211 rows total: N = 119, mean 27.0 assessments per participant). Columns: `user_id` (anonymous code P001–P119), `exam_period` (assessment window label: `baseline` / `pre_exam` / `post_exam` / `NA` for routine assessments), five EMA-based PID-5 domain scores (`pid5_negative_affectivity` etc., sum of 3 items, range 0–9), and five baseline full PID-5 domain scores (`domain_*_baseline`, sum scores from the 220-item questionnaire). Ten participants have `NA` for all baseline domain scores (did not complete the T1 questionnaire); these participants are excluded from the LOO comparison (script 05) but included in all other analyses.

See `data_dictionary.pdf` for full variable descriptions and scale ranges.

## Reproducing the Analyses

Run scripts in numerical order. Each script reads from `data/processed/` and/or prior results bundles, and writes to `results/` (created automatically).

---

### Script 00 — Descriptive statistics and reliability

```r
source("scripts/00_descriptives_and_reliability.R")
```

Produces all descriptive and reliability tables reported in the Supplementary Materials (Tables S1–S8):

- **Table S1**: Internal consistency of baseline PID-5 domains (α and ω, full and attention-check-cleaned samples). Values are hard-coded from the original analysis; the 220-item questionnaire item-level data are not included in this repository (available from the corresponding author).
- **Table S2**: EMA multilevel reliability following Lai (2021): α₂L, αB, αW, ω₂L, ωB, ωW. Values are hard-coded; to recompute from item-level data, add the individual EMA item columns (`pid5_1`–`pid5_15`) to `pid5_required_columns.csv`.
- **Table S3**: Convergent validity correlations between EMA domain means and baseline domain scores.
- **Table S5/S6**: Voice outcome descriptives by assessment period and within-person variability.
- **Tables S7/S8**: EMA PID-5 domain descriptives (between-person and within-person variability).

**Output:** `results/descriptives/table_s*.csv`

### Scripts 01–02 — F0 moderation model (Tables 2, S9–S11)

```r
source("scripts/01_prepare_f0_data.R")   # ~1 min
source("scripts/02_fit_f0_moderation.R") # ~15–25 min
```

**Script 01** reads both data files, computes per-participant mean F0 across vowels /a/, /i/, /u/, codes the two orthogonal within-person contrasts (stress: PRE vs BASELINE; recovery: POST vs PRE), applies within-subject mean imputation for the single sporadic missing EMA value, and standardises EMA domain scores for the Stan measurement model. Saves `results/F0/stan_bundle_f0.rds`.

**Script 02** fits `stan/F0/f0mean_pid5_moderation.stan` — a joint measurement-and-outcome model that simultaneously estimates latent PID-5 trait scores from repeated EMA observations and their moderation of F0 stress responses. Sampler settings: 4 chains × 6 000 post-warmup draws, `adapt_delta = 0.99`, `max_treedepth = 15`. Saves the fitted object, posterior summary, PPC plot, and moderation probability table (Table 2).

**Key results to verify:**

| Parameter | Median | 95% CrI | PD |
|-----------|--------|----------|----|
| β₁ stress effect | 3.27 Hz | [0.81, 5.71] | 0.995 |
| γ₁ Neg. Affectivity × stress | 3.14 Hz | [0.37, 5.89] | 0.97 |
| γ₂ Antagonism × recovery | 3.16 Hz | [0.51, 5.78] | 0.97 |

**Output:** `results/F0/`

### Script 05 — LOO-CV model comparison (Tables 4–5)

```r
source("scripts/05_loo_comparison.R") # ~2–3 hours (fits 2 models)
```

Compares three approaches to measuring PID-5 traits in predicting F0 stress trajectories using leave-one-out cross-validation (ELPD, LOOIC):

- **Model A (EMA-only)**: latent traits estimated from repeated EMA assessments via the joint measurement model (`stan/LOO/f0mean_pid5_moderation_improved.stan`)
- **Model B (Baseline-only)**: standardised baseline PID-5 scores as fixed predictors (`stan/LOO/pid5_baseline_moderation_improved.stan`)
- **Model C (Combined)**: both EMA latent traits and baseline fixed predictors (`stan/LOO/pid5_ema_plus_baseline_moderation_improved.stan`)

**Sample restriction:** 10 of 119 participants lacked baseline PID-5 data; this script runs on N = 109 for all three models to ensure a fair comparison.

**Key result to verify (Table 4):** All three models show comparable out-of-sample predictive accuracy (|Δ ELPD| < 2 SE). Table 5 shows that the EMA-based estimate of Negative Affectivity × stress moderation has ~28% narrower credible intervals than the baseline-only estimate despite comparable point estimates.

**Output:** `results/LOO/`

### Script 06 — Within-person temporal covariation (Supplementary p. 17)

```r
source("scripts/06_temporal_covariation.R") # ~2–4 hours
```

Examines whether momentary PID-5 state fluctuations (EMA assessments linked to each voice recording session via `exam_period`) covary with concurrent F0. Compares two hierarchical specifications via LOO-CV:

- **Baseline model**: population-average fixed slopes + random intercepts (`stan/temporal/baseline_model.stan`)
- **Random slopes model**: person-specific within-person PID-5–F0 slopes (`stan/temporal/random_slopes_noncentered.stan`)

**Important note on scaling:** `X_wp` predictors are passed to Stan in raw (non-standardised) units. Standardising would cause the random slopes model to absorb nearly all residual error with only 3 observations per person (R² ≈ 1); raw units preserve the regularisation effect of the `sigma_beta ~ Normal(0, 2)` prior.

**Key results to verify (Supplementary p. 17):** Random slopes model R² ≈ 35–44% vs baseline ≈ 2.5–3%; ELPD difference ≈ 40–76 points. Numerical results in this repository are approximately consistent with manuscript values (see note below).

**Output:** `results/temporal/`

---

### Script 07 — Slope heterogeneity analysis (Supplementary p. 17)

```r
source("scripts/07_slope_heterogeneity.R") # ~5–10 min
```

Requires `results/temporal/fit_random_slopes.RDS` (run script 06 first). Extracts individual-level total slopes (fixed + random) for all 119 participants × 5 domains, classifies each by whether the 95% credible interval excludes zero, and summarises between-subject slope variability (σ_β per domain).

**Key results to verify:**

| Statistic | Manuscript | Script |
|-----------|-----------|--------|
| CI spanning zero | 99.7% | ~99.5% |
| σ_β range (median) | not specified | 2.95–4.59 Hz |

**Output:** `results/temporal/individual_slopes_summary.csv`, `heterogeneity_summary.csv`, `sigma_beta_summary.csv`

## Stan Models

### `stan/F0/` and `stan/NNE/` — Joint measurement-and-outcome models

Both implement the same two-component architecture described in the Supplementary Materials (Statistical Models section):

1. **Measurement model**: each participant's ~20 EMA observations per domain are noisy indicators of a latent trait θ_{i,d}; occasion-level variability σ_ema_d is estimated jointly.
2. **Outcome model**: hierarchical linear regression with random intercepts and slopes for both contrasts, plus 10 cross-level interactions (5 PID-5 domains × 2 contrasts).

Models differ only in intercept prior (F0: `Normal(220, 30)` Hz; NNE: `Normal(-27, 3)` dB) and residual SD prior.

### `stan/LOO/` — Three measurement comparison models

- `f0mean_pid5_moderation_improved.stan`: EMA latent-trait model (improved parametrisation)
- `pid5_baseline_moderation_improved.stan`: fixed-predictor baseline model (no measurement model layer)
- `pid5_ema_plus_baseline_moderation_improved.stan`: combined model

### `stan/temporal/` — Within-person covariation models

- `baseline_model.stan`: fixed within-person slopes + random intercepts
- `random_slopes_noncentered.stan`: person-specific slopes with non-centred parametrisation

## Convergence Criteria

All models should show:
- No divergent transitions
- R̂ < 1.01 for all parameters
- Bulk-ESS > 1 000 for key parameters

Convergence diagnostics are saved to `results/*/convergence_summary.csv` and printed to console. The temporal covariation random slopes model may show elevated Pareto-k values in LOO (expected with N = 3 per person); this does not affect the R² comparison.
